    Getting Started with CrydenSync

    📦 Installation Methods

    Method 1: Go Get (Recommended)

    
                        go get github.com/crydensync/cryden@v1.0.0

    Verify

    
                        package main

                        import (
                            "fmt"

                            "github.com/crydensync/cryden"
                        )

                        func main() {
                            engine := cryden.New()
                            fmt.Printf("Cryden v1.0.0 ready: %T\n", engine)
                        }

    Method 2: Clone and Build

    
                        # Clone the repository
                        git clone https://github.com/crydensync/cryden.git
                        cd cryden

                        # Build the library
                        go build ./...

                        # Run the complete example
                        cd examples/complete
                        go run main.go

                        # Or try the basic example
                        cd ../basic
                        go run main.go

                        # Run tests
                        cd ../..
                        go test ./internal/tests/... -v

    Method 3: Download Binary

    
                        # Download latest release
                        wget https://github.com/crydensync/cryden/releases/latest/download/cryden-linux-amd64.tar.gz
                        tar -xzf cryden-linux-amd64.tar.gz
                        ./cryden -version

    Versioning

    Cryden follows Semantic Versioning:

    • v1.0.0 - Stable core API
    • v1.1.0 - New features (gRPC, CLI)
    • v2.0.0 - Breaking changes

    Module Proxy

    If you're behind a firewall, use Go proxy:

    
                        export GOPROXY=https://proxy.golang.org.direct
